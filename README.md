Descripción del proyecto

EcoHuella es una aplicación móvil nativa para Android creada como una herramienta que pueda cumplir con pequeños impulsos hacia la sostenibilidad ecológica. Una solución tecnológica para transformar la conciencia y hábitos de las personas. 
La función principal es registrar acciones ecológicas diarias de los usuarios de manera que se visualice un progreso a través de un sistema de puntos que fomente la creación de hábitos que reduzcan la huella de carbono personal. 
Exposición del problema 
Con esta aplicación se espera dar un paso en el avance para aquellas personas que desean llevar una vida más sostenible y que contribuya con pequeñas acciones con el medio ambiente y que no saben de qué manera realmente pueden generar un cambio. 


Plataforma

•	Sistema operativo: Android

•	Versión: Mínima 7.0 para garantizar compatibilidad con diversos dispositivos.

•	Lenguaje: Kotlin utilizando Android Studio.

•	Almacenamiento: SharedPreferences que almacenará datos simples. 


Interfaz de usuario e interfaz de administrador

La interfaz de usuario será vista principal de la aplicación. Una pantalla de inicio que muestre el puntaje acumulado y un mensaje relacionado con los puntos ganados. También se presentará un menú de lista donde el usuario marcará las tareas completadas.
La interfaz de administrador puede estar dentro de la aplicación, pero de manera protegida. En esta sección se podrá gestionar las actividades con la capacidad de añadir nuevas tareas o cambiar el valor de los puntos. También se podrá borrar las estadísticas. 

Funcionalidad

La funcionalidad de la aplicación seguirá los siguientes pasos:
1.	Registro de tareas: El usuario selecciona las actividades que realizó
2.	Cálculo: Se suma los puntos por cada tarea.
Diseño (Wireframes o esquemas de página)
La primera pantalla de Eco-Huella muestra la interfaz principal que visualizará el usuario, se representan la lista de tareas que debe cumplir y muestra los puntos obtenidos durante el día. En la segunda pantalla se detalla las estadísticas del usuario a lo largo de la semana con un gráfico de barras. La última pantalla se definen los logros obtenidos junto con una función de activar un mensaje de recordatorio al usuario.

Conclusión

Este proyecto puede demostrar la aplicación de manera sencilla y práctica de la parte técnica de una aplicación móvil y la lógica de la programación con el propósito ambiental que apunta a crear hábitos responsables. Con esta aplicación presenta los requisitos básicos para un software funcional y a la vez proponer una solución para uno de los desafíos de la sociedad.
